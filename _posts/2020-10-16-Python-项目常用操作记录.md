#### 生成requirements.txt

```markdown
# 安装
pip install pipreqs
# 在当前目录生成
pipreqs . --encoding=utf8 --force

# 使用pip安装requirements.txt文件
pip install -r requirements.txt

# 使用conda安装requirements.txt文件
conda install --yes --file requirements.txt
```



#### 项目封装

```markdown
pip install pyinstaller
pyinstaller -F XXX.py
```
