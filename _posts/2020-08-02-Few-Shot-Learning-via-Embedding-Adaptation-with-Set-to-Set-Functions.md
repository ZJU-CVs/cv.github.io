### 1. Introduction

- 目前许多少样本学习方法通过可见类学习一个实例嵌入函数，并将其应用于标签有限的不可见类的实例中。但是这种迁移方式的任务无关的：相对于不可见类，嵌入函数的学习不是最佳的判别式，因此会影响模型在目标任务的性能
- 本文通过一个*set-to-set*函数使实例嵌入与目标分类任务相适应，产生具有任务特定性和区分性的嵌入。
