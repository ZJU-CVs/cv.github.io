### 已有项目上传

```bash
git init
git add .
git commit -m 'first_commit'
git remote add origin https://XXX.git
git push -u origin main
```



### 更新(上传)

```
git add .
git commit -m "first commit"
git push -u origin master
```



### 版本找回

```
git fsck --lost-found
git reset --hard [hash]
```
