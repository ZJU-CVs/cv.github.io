#### MySQL

```bash
# 安装
brew install mysql # mac系统可用brew下载

# 直接进入官网下载安装包
https://dev.mysql.com/downloads/mysql/

# 启动MySQL
mysql.server start

# 关闭MySQL
mysql.server stop

# 查看状态
mysql.server status
```



#### postgresql

```bash
pg_ctl -D /usr/local/var/postgres -l logfile start

pg_ctl -D /usr/local/var/postgres -l logfile stop
```
