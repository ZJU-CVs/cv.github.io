#### MySQL

```bash
# 安装
brew install mysql

# 启动MySQL
mysql.server start
 
# 关闭MySQL
mysql.server stop
```



#### postgresql

```bash
pg_ctl -D /usr/local/var/postgres -l logfile start

pg_ctl -D /usr/local/var/postgres -l logfile stop
```



#### mangodb

```bash
mongod -dbpath /Users/jiangyu/mangodb/data
```
